# Alimer Engine
**Alimer Engine** is a cross platform 2D and 3D engine implemented in C++ and released under Apache 2.0 license, inspired by [Urho3D](https://github.com/urho3d/Urho3D) and [Ogre3D](http://www.ogre3d.org).
